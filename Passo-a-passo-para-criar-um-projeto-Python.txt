1° Criar uma 'VENV' 
 - python -m venv venv

2° Ativar a venv
 - entrar na venv e depois na pasta Scripts e execultar o activate

3° Depois de ativar o activate, voltar para a raiz do projeto

4° Instalando o django
 - pip install django

5° Criando projeto django
 - django-admin startproject gestao_rh .

6° Subindo projeto
 - python manage.py runserver

7° Criar um modulo
 - python manage.py startapp empresa

8° Criando migrações ou JPA
 - python manage.py makemigrations

9° Criando migrações
 - python manage.py migrate

10° Criando superuser
 - python manage.py createsuperuser
